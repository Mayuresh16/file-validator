{# Naming Convention Notes #}
💡 Common Naming Conventions:
  • Environment Prefix: td_ (Teradata), bq_ (BigQuery), prod_, dev_
  • Date Format: YYYYMMDD, YYYY-MM-DD, YYYYMMDD_HHMMSS
  • Table/Dataset Name: Usually the main identifier in the middle portion
  • File Extensions: .csv, .txt, .dat, .TXT
